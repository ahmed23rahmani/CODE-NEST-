Instructions de Personnalisation

Pour appliquer une variante de couleur, modifiez les variables CSS dans le fichier styles.css : 

:root {
    /* Modifier ces valeurs pour changer la palette */
    --color-nuit: #071229;      /* Arrière-plan principal */
    --color-profond: #0f2b45;   /* Éléments secondaires */
    --color-gris: #1f3b55;      /* Éléments tertiaires */
    --color-accent: #2EA6FF;    /* Couleur d'accent */
    --color-white: #FFFFFF;     /* Texte et éléments clairs */
}

Les changements s'appliqueront automatiquement à toute la maquette grâce à l'utilisation des variables CSS.
